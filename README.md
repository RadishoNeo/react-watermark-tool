# <img src="public/logo.svg" width="40" height="40" align="center" /> Watermark Pro

**Watermark Pro** is a next-generation, high-performance web tool designed for creators to secure their digital work with professional-grade watermarks. Built with a focus on privacy, aesthetics, and user experience.

## ‚ú® Features

- **Next-Gen UI**: A stunning, futuristic interface using glassmorphism, mesh gradients, and smooth micro-animations.
- **Privacy First**: All image processing happens 100% locally in your browser. Your images are never uploaded to any server.
- **High Performance**: Native Canvas-based rendering engine ensuring lightning-fast performance and high-resolution output.
- **Real-time Customization**:
  - Adjust text content, font size, and color.
  - Control rotation angle and opacity.
  - Fine-tune tiling gap and offsets for perfect coverage.
- **Full Responsive**: Works seamlessly across desktops, tablets, and mobile devices.

## üõ†Ô∏è Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/RadishoNeo/react-watermark-tool.git
   ```

2. Navigate to the project directory:

   ```bash
   cd react-watermark-tool
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

## üìñ Usage

1. **Upload**: Click the central zone or drag-and-drop an image.
2. **Edit**: Use the right sidebar to customize your watermark settings.
3. **Preview**: See changes instantly in the interactive preview area.
4. **Export**: Click "Download Result" to save your watermarked image locally.
